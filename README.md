# Python Exercises

Collection of small **Python** practice scripts created in 2024 and still actively updated to review concepts and gradually increase difficulty.  
Scripts range from very basic operations to slightly more structured tasks.

---

## Tech Stack
- **Programming language:** Python
- **IDE:** PyCharm

---

## Main Features
- Basic arithmetic operations
- List filtering and simple statistics
- Duplicate-removal utilities
- Simple calculators (tips, totals, receipts)

*Project created for learning and portfolio purposes.*
